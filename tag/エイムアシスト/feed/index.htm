ディープラーニングによる物体検知技術を用いたアシストツールです。

APEXの画面を見て、一度は「今のAIなら敵を認識したチートを作れるんじゃないの？」と思ったことはあると思います。

それを現実化したのが、敵認識アシストになります。

実際に使ってみると以下のように結構吸い付いているのがご覧いただけると思います。

 

yolo というのは、リアルタイム映像から 人・車・物体 など様々な検出を可能にする機械学習アルゴリズムのことです。

yoloの本来の使い方

この技術をAPEXゲーム用に改良したものが、今回ご紹介する AI型 敵認識エイムアシストです。

実際に使用してみると、下記の状況では yoloは 86% の確率で 敵だと認識しています。

敵を認識した場合は、その BOX の中心にマウスが動く様に、 GHUB の仮想マウスが動きます。

これにより、敵を狙わなくても 敵を認識した yolo がターゲット補足を支援してくれます。

他のエイムアシストツールとの違い

以前、APEXのエイムアシストツールというのは、AHK(オートホットキー) という自動スクリプトや、色認識というのが主流でした。

はっきり言いますが、それらは全て詐欺ツールです。

色認識は、赤色に反応して マウスを入力するというものですが、apexの敵の色は赤色でもなんでもないです。

しかし、訓練所のダミー人形は赤色です。　なので 訓練所の人形にだけ吸い付くソフトが完成します。

この訓練所で吸い付いている動画を PR として 色認識を販売している方が大勢いましたが、使い物にならないとわかっている製品を売ってる時点で詐欺です。

この記事をお読みになっている方の中にもそういったゴミツールをつかまされた方がいるのではないでしょうか。

AI アシストの弱点

設定はある程度カスタマイズ可能ですが、やはり エイムボットを搭載したチートに比べると劣ります。

AIによる物体認識なので、当然、人間の目でも見つけにくい遠方の敵や、オブジェクトとほぼ重なっている敵、　見えにくいところにハイドしている敵は検出できない場合があります。

本格的なカスタマイズと、遠方の敵にも安定的な吸い付きが欲しい方は チート を検討するべきだと思います。

また、このツールは更新されていないので今後検出される可能性はあります。　それが1ヶ月後なのか、1年後なのかは誰にもわかりません。

「エイムアシスト」型のチートとなります。
エイムアシストとグロー(WH) の２機能のみでコンパクトなセットです。
危険でバレやすいエイムボットに飽きて、正当なプレイをしたい方向けとなっています。
設定のカスタマイズはできないため、このチートは熱くなって設定を強めてしまう心配はありません。
少しブーストされていながら、切り札はないので、他のプレイヤーと同じ条件で戦うことができます。

CS PAD のエイムアシストは 0.6 程度と言われていますが、 SM-APEXは 体感ですが、
0.7 〜 0.8 ほどの強さを感じますので、 CS PAD より強力です。
しかし、エイムは下記動画のように驚くほど自然です。

エイムアシストは最大40mまで有効

観戦通報のためだけでなく、サーバーサイドチェックにも対しても有効です。
Aimbotではなく、AIM Assist です。ご自身のaimも必要です。
